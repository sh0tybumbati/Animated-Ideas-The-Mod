# Project Plan - GarrettTheCarrotMod

## Technical Roadmap
1.  **Phase 1: Research & Selection** (Current)
    - Categorize features by difficulty and impact.
    - Choose Fabric API for implementation.
2.  **Phase 2: Environment Setup**
    - Initialize the Gradle project.
    - Set up assets (textures, models).
3.  **Phase 3: Core Implementation**
    - Implement the "Part 1" features as the MVP.
4.  **Phase 4: Chaos & Specials**
    - Add "Worst Ideas" and themed specials as optional configs.
