# GarrettTheCarrotMod

A Minecraft mod inspired by [GarrettTheCarrot's](https://www.youtube.com/@GarrettTheCarrot) "Animating your Minecraft Ideas" series. This mod aims to bring the creative (and sometimes absurd) viewer-submitted ideas to life within Minecraft.

## Status
**Phase:** Planning
This project is currently in the architectural and brainstorming stage.

## Goals
- Implement features based on viewer-submitted ideas featured in GarrettTheCarrot's animations.
- Maintain a balance between "Good Ideas" (useful features) and "Worst Ideas" (chaotic/funny features).
- Provide a modular experience where players can toggle specific ideas.
